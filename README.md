# wetterstation

Seit Dezember 2006 werden Außenlufttemperatur und Luftfeuchte, Globalstrahlung, Regen und Luftdruck von der Wetterstation der Schule aufgezeichnet.

- Außenlufttemperatur  (°C)
- Außenluftfeuchte     (%)
- Globalstrahlung      (W/m²)
- Luftdruck            (hPa)
- Regen                (mm/m²)
